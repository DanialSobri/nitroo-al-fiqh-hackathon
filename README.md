# al-fiqh-hackathon

## Neo AI - Next‑Gen Optimized Advisor, driven by Agentic AI

Neo AI is a cutting-edge Retrieval Augmented Generation (RAG) system designed for Islamic finance and Shariah compliance. It continuously gathers official documents from multiple trusted sources, organizes them in a vector database, and delivers precise answers through an intelligent Q&A interface powered by advanced language models. Built to stay current, Neo AI ensures there’s no knowledge cutoff — keeping you aligned with the latest Shariah regulations at all times.

## At Tathqeeb

Tathqeeb is an AI-powered Shariah compliance agent that never rests. It continuously monitors your contracts, ensuring they stay aligned with the latest Shariah regulations the moment new updates are introduced. Powered by the Neo AI engine, it delivers real-time compliance oversight with precision and reliability.
